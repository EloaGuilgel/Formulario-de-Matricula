# 📄 Formulário de Matrícula

![Capa do projeto](assets/Formulario.png.png)

Projeto de um **Formulário de Matrícula** desenvolvido utilizando **HTML e CSS**, com foco em praticar a criação de formulários, organização de layout e estilização com CSS.

O objetivo do projeto é simular o cadastro de um aluno em uma instituição, trabalhando diferentes tipos de campos de formulário e aplicando boas práticas de estruturação de layout.

---

## 🚀 Tecnologias utilizadas

Este projeto foi desenvolvido com as seguintes tecnologias:

- **HTML5**
- **CSS3**

---

## 📚 O que foi praticado

Durante o desenvolvimento deste projeto foram praticados conceitos importantes como:

- Estruturação semântica com **HTML**
- Criação de **formulários**
- Uso de diferentes tipos de **inputs**
- Organização de layout com **CSS**
- Estilização de formulários
- Separação de responsabilidades entre **HTML e CSS**
